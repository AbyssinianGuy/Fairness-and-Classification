# Fainess-and-Classification
This is an attempt to increase the accuracy of a prediction by eliminating features that can lead to bias.

Classifiers used in this test are:
1. SVM
2. Random Forest (RF)
3. Decision Tree
4. K-nearst neighbor
5. Boosting Classifier
